# Hello World – GitHub Tutorial

Repository URL: https://github.com/ladino72/hello-world

Este repositorio forma parte del ejercicio del curso "Inteligencia Artificial para uso en el aula de clase".  
Su propósito es practicar el flujo básico de trabajo con GitHub: ramas, commits, pull requests y documentación.

En este repositorio seguiré el tutorial oficial "Hello World" de GitHub y usaré este espacio para probar pequeños cambios, registrar mis avances y familiarizarme con las herramientas de colaboración.


## Reflexión personal

Al crear este README entendí la importancia de documentar de manera clara y sencilla lo que hago.  
No es solo un requisito técnico: es una forma de comunicar, de organizar mis ideas y de hacer que el proyecto sea accesible para cualquiera.  
Este ejercicio me permitió aplicar lo aprendido en el curso y valorar cómo una buena documentación facilita el trabajo colaborativo y el aprendizaje continuo.
